# Breaking Bad Character

A demo flutter app with Breaking Bad Api

### Framework
    - Flutter

### Architecture
    - Bloc for State Management
    - Repository pattern

### Flavor
    - Staging
    - Production

### How to run Flavor
  - Run Staging 
  

    `flutter run --flavor staging -t lib/main_staging.dart`


- Run Production


    `flutter run --flavor production -t lib/main.dart`
